# ScriptGuru

## Installation

### Install soft
1. `sudo apt-get update`
1. `sudo apt-get install nodejs`
1. `sudo apt-get install npm`
1. `sudo npm install -g gulp`

### Install project
1. `git clone https://github.com/nikitapavets/script-guru.git`
1. `cd ./script-guru`
1. `npm install`
1. `gulp build`
